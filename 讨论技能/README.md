# 讨论技能

这个目录不是主仓正文的一部分，而是未来可独立抽出的讨论技能种子。

它的任务很窄：

- 用户提问后，先定位对应章节；
- 只读取最小必要材料；
- 回答时区分"文本已有结论""从文本推出的推论""超出文本的扩展提案"；
- 如果材料不足，直接说明不足，不脑补。

## 目录说明

- `SKILL.md`：技能总说明与回答流程。
- `references/章节映射.md`：问题类型到阅读材料的映射。
- `references/回答规则.md`：回答格式与防错规则。
- `references/仓库阅读顺序.md`：材料优先级。
- `agents/openai.yaml`：可供外部系统读取的最小 agent 元信息。

## 抽离原则

- 主仓只保留这个种子目录，不把它和主仓发布周期绑死。
- 真正独立成仓时，优先补 `README`、版本策略、验证脚本和最小示例，而不是继续堆概念说明。
