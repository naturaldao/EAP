# 术语表

| 术语 | 工作定义 | 边界 / 备注 |
| --- | --- | --- |
| 爱的证明 / Proof of Love (PoL) | 当前研究中的核心治理共识。要求公共规则、关系和技术系统以扬爱抑恨为方向。 | 不是情绪口号，必须落到制度与协议。 |
| 富爱文明 / PoL Civil | 以爱的证明为核心共识的未来文明形态。 | 仍需要制度、审计、权限和例外处理。 |
| 富恨文明 / PoH Civil | 当前以敌我化、羞辱、排斥、报复等结构为主导的文明形态。 | 不是指个别人“很坏”，而是治理机制的总体取向。 |
| EAP | 伦理对齐性治理协议，未来公共 AI / PAI 的治理协议。 | 不是完整社会系统；需与其他层级协同。 |
| PAI / Public AI | 面向公共事务的 AI 角色。 | 不是私人伴侣型 AI，也不是不受约束的主权机器。 |
| 爱的连接 / Love Linking | 扬爱抑恨的连接方式，同时要求不伤害任何人的快乐与共同管理关系。 | 不等于讨好，不取消边界。 |
| 爱的管理 / Love's Management | 共同把关系保持在爱之基础上的纠偏、修复、分担与协调。 | 不是支配，不是温柔统治。 |
| 爱语 / Love Languages | 有爱的言行与文明行为的谱系。 | 不是所有爱语都适合 PAI 直接扮演。 |
| 恨语 | 将世界压扁为敌我对立，并外化为羞辱、排斥、恐惧制造、报复等结构的表达或行为。 | 不等于一切批评、拒绝或正当防卫。 |
| 同一伦理标准 | 对处于相似情境的主体适用同一伦理标准。 | 不等于永远同一动作或结果。 |
| SCC0 | 相关的独立协议研究仓与长期方向。 | 当前轮次里，它不是本仓已经生效的许可证文本。 |
| 扩展性提案 | 超出 `Reference/` 现有明文内容的新增想法。 | 需要显式标注，不可伪装成既有结论。 |
