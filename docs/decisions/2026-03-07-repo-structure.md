# ADR 2026-03-07：仓库结构分层

## Status

Accepted

## Context

当前仓库同时承载锚点文本、工作稿、评论稿和实现思考，长期下去会让不同稳定度的内容互相污染。

## Decision

- `Reference/` 只承载相对稳定的核心锚点文本。
- `proposal/` 承载工作稿、评注信、实验性整合稿和外置项目 seed。
- `docs/` 承载实现层文档、路线图、术语表、失败模式和决策记录。
- 未来 discussion skill 不并入主仓运行资产，只在当前仓保留 seed。

## Consequences

- 读者能更快区分“原理文本”和“工作草案”。
- 新的实现层分析不会再被误读成 `Reference/` 既有立场。
- 主仓可以继续保持研究强度，而不必假装自己已经是成品软件仓。
