# ADR 2026-03-07：当前轮次不切换仓库许可证

## Status

Accepted

## Context

团队长期方向更倾向 `SCC0`，且 `../SCC0-License` 已作为独立研究仓存在。但当前 EAP 回合尚未完成，本仓 `LICENSE` 仍是当前已生效文本。

## Decision

- 当前轮次只在文档中准确说明与 `../SCC0-License` 的关系。
- 不在本轮修改本仓许可证文本，不把 `SCC0` 写成已经生效的许可证。
- 等当前 EAP 回合稳定后，再单独处理许可证切换与仓间对位。

## Consequences

- 避免在协议尚未稳定时引入法律文本层面的事实错误。
- 让 EAP 与 SCC0 的对位成为下一阶段的明确任务，而不是当前轮次里的模糊口号。
